# Election

Development Environment
------
Mac OSX 10.10.4
Ruby 2.2.0p0
Rails 4.2.1

Running Locally
------
```bash
git clone https://github.com/SmallNic/election-0ptimus.git
cd election-0ptimus
bundle install
rake db:create
rake db:migrate
rails s
```
